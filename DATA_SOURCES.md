## Nutrition Data

The nutrition pipeline uses a **unified index** merged in-memory from 3 sources. Priority on duplicate English names: CIQUAL > OpenNutrition > MEXT.

### OpenNutrition (baseline)

| Field | Value |
|---|---|
| **Source** | OpenNutrition (https://www.opennutrition.app/) |
| **License** | ODbL 1.0 (Open Database License) — requires attribution and Share-Alike |
| **File** | `server/packages/recipe_scraper/src/recipe_scraper/data/opennutrition_index.json` (~5,300 entries) |
| **Content** | Food items with macros per 100g (kcal, protein, fat, carbs, fiber, sugar, saturated fat) aggregated from USDA, CNF, FRIDA, AUSNUT. Mineral data (Ca, Fe, Mg, K, Na, Zn) backfilled from CIQUAL via deterministic name matching (`backfill_minerals_fast.py`). |
| **Used by** | `nutrition_matcher.py` — merged into unified nutrition index |
| **Last updated** | 2026-02-09 |

OpenNutrition incorporates data from [Open Food Facts](https://world.openfoodfacts.org/) (ODbL). Attribution: © Open Food Facts contributors.

The OpenNutrition dataset is also available as a MCP server: https://github.com/deadletterq/mcp-opennutrition (GPL-3.0).

### CIQUAL 2025 (French food composition)

| Field | Value |
|---|---|
| **Source** | ANSES — Table de composition nutritionnelle Ciqual 2025 |
| **License** | Licence Ouverte 2.0 (Etalab) |
| **URL** | https://ciqual.anses.fr/ — Data: https://entrepot.recherche.data.gouv.fr/dataset.xhtml?persistentId=doi:10.57745/RDMHWY |
| **File** | `server/packages/recipe_scraper/src/recipe_scraper/data/ciqual_index.json` (~3,500 entries) |
| **Content** | French and English food names with per-100g macros (kcal, protein, fat, carbs, fiber, sugar, saturated fat) and minerals (calcium, iron, magnesium, potassium, sodium, zinc in mg). Strong coverage of European/French foods (burrata, pancetta, liqueurs, breadcrumbs). |
| **Used by** | `nutrition_matcher.py` — merged into unified nutrition index (highest priority on duplicates) |
| **Generated by** | `server/scripts/import_nutrition_sources.py --ciqual-xml --ciqual-xlsx` |
| **Last updated** | 2026-02-22 |

### MEXT Japan — Standard Tables of Food Composition in Japan, 2015 (7th Revised Edition)

| Field | Value |
|---|---|
| **Source** | Ministry of Education, Culture, Sports, Science and Technology (Japan) |
| **License** | Government of Japan Standard Terms of Use (open data) |
| **URL** | https://www.mext.go.jp/en/policy/science_technology/policy/title01/detail01/1374030.htm |
| **File** | `server/packages/recipe_scraper/src/recipe_scraper/data/mext_index.json` (~2,200 entries) |
| **Content** | English food names with per-100g macros and minerals (calcium, iron, magnesium, potassium, sodium, zinc in mg). Strong coverage of Asian ingredients (mirin, natto, miso varieties, tofu, sesame, seaweed). No sugar column. Note: 7th edition (2015) has been superseded by the 8th edition (2021); this project uses 7th edition data. |
| **Used by** | `nutrition_matcher.py` — merged into unified nutrition index (lowest priority on duplicates) |
| **Generated by** | `server/scripts/import_nutrition_sources.py --mext-xlsx` |
| **Last updated** | 2026-02-22 |

### USDA FoodData Central API (mineral backfill)

| Field | Value |
|---|---|
| **Source** | USDA FoodData Central (https://fdc.nal.usda.gov/) |
| **License** | CC0 1.0 Universal (Public Domain) |
| **API** | `https://api.nal.usda.gov/fdc/v1/food/{fdc_id}` |
| **Content** | Full nutrient profiles for individual foods, queried by FDC ID |
| **Used by** | `backfill_resolved_minerals.py` — fetches mineral data (Ca, Fe, Mg, K, Na, Zn) for ingredients in `resolved_ingredients.json` that have a `fdc_id` |
| **Last updated** | 2026-02-27 |

This is not a static dataset but an API used to enrich resolved ingredients with mineral data. The results are stored in `resolved_ingredients.json` alongside the existing macro data.

## Macronutrient Reference Ranges

The Meal Planner uses macronutrient distribution ranges to evaluate meal balance.

| Source | Protein | Carbs | Fat | Notes |
|---|---|---|---|---|
| **ANSES (France, 2016)** | 10-20% | 40-55% | 35-40% | Primary reference (French audience) |
| IOM/USDA AMDR | 10-35% | 45-65% | 20-35% | US reference, broader ranges |
| EFSA (Europe) | ~10-15% | ~52% | ~31.5% | Based on labelling RIs |
| WHO/OMS | — | — | <= 30% | Focus on fat quality, not strict ranges |

The app uses **ANSES values** as default, with ideal targets at the midpoint of each range.
See `MacroBalanceBar.jsx` for implementation details.
