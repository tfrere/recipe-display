
Titre : Ramen

Note : Pas traditionnel du tout, c’est juste pour Léa

Pour 2 personnes
Ingrédients :
100g de lard fumé ( désossé, coupés en tranches assez fines )
2 oignons nouveaux ( haché finement, séparer le vert des bulbes )
3 petits pak choi ( séparer les feuilles des branches, couper les branches en deux dans la longueur )
1 gousse d’ail ( haché finement )
2 oeufs
Dashi ( ou poisson bonito )
sauce soja
nouilles udon ( grosses nouilles blanches )
1 l d'eau 
gingembre frais ( finement haché )
algue nori ( en petit carrés de 3cm par 3cm, toppings optionnel )
oignons frits ( toppings optionnel )
coriandre fraiche ( hachée, toppings optionnel ) 

Préparation :
Oeufs marinés ( 24-48h ) : 
Faire cuire les oeufs pendant 6-8 minutes 
Les faire mariner dans un mélange sauce soja + dashi + eau 
Laisser mariner 24-48h 
           Pak choi :
Faire blanchir le pak choi ( mettre les feuilles en dernier pour une cuisson uniforme )
Le faire dorer qq minutes à la poêle avec l’ail dans de l’huile de sésame et réserver. 

Bouillon :
Faire cuire le lard jusqu'à coloration dans un tout petit peu d’huile de sésame
ajouter oignons, gingembre et cuire 3 minutes en remuant
Ajouter l’eau, porter à ébullition puis faire mijoter pendant 30 minutes ( tu peux ecumer si necessaire, ca eclairci le bouillon )
À convenance : Ajouter 1 càs de sauce soja et un petit peu de dashi si le goût du bouillon n’est pas assez prononcé

	Nouilles : 
Quand tout est prêt :  porter le bouillon à ébullition et faire cuire les nouilles suivant les indications du paquet

Dressage : 
Dans un grand bol
Remplir de bouillon
Ajouter une poignée de nouilles
Ajouter le pak choi
Couper l’oeuf en deux et disposer les deux parties ensemble sur le dessus du bouillon ( jaune vers le haut ) 
Toppings :
tige hachée de l’oignon nouveau
oignons frits
algue nori 
coriandre hachée
